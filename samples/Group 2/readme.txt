The next directory contains the digitized samples from the experiments 2 & 3. 

Samples were created with MX4 proof profile from GMG ColorProof Software, with
synthetic noise added to simulate graininess. 30 samples with different noise
levels were printed using the Epson Stylus Pro 4880 printer in a CGS ORIS Semi
matte paper. Samples were digitized with Heidelberg Linoscan 2650 scanner with 
software SilverFast version v6.6.2r5 in RGB and stored in .tif format. CIELab 
transformations were performed using Adobe Photoshop CC 2019.

Sets Description:
-----------------
320-Chart		- Samples containing the 320-Chart test chart defined 
			  in ISO/IEC 24790.
MediaWedge		- Samples containing the FOGRA MediaWedge CMYK V3.0a.


Directory Structure:
--------------------
320-Chart
 └─ LAB_600_DPI			- CIELab samples measured at 600 dpi in .tif format. 
MediaWedge
 ├─ LAB_600_DPI			- CIELab samples measured at 600 dpi in .tif format.
 └─ RGB_150_DPI			- RGB samples measured at 150 dpi in .png format.
